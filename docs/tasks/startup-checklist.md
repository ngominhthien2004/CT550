# Startup Checklist (One-time)

Mục tiêu: checklist dùng 1 lần khi bắt đầu project/module lớn. Sau khi hoàn tất, giữ file này làm baseline.

## Baseline
- [x] Use one git repository at project root.
- [x] Keep monorepo folders: `backend/`, `frontend/`, `docs/`.
- [x] Ensure docs folders: `docs/diagrams`, `docs/report-templates`, `docs/tasks`.
- [ ] Ensure config files (e.g., `.gitignore`, `.vscode`) generated by sub-frameworks (Vite/Node) are moved to the monorepo root and updated.
- [x] Decide Docker timing (defer unless deployment/team parity requires it).
- [x] Confirm local run/validation commands for backend and frontend.
- [x] Setup `gitnexus`, create initial commit, and push the base project to the remote repository.
