# GIF Upload Policy

- Animated posts: GIF only. The server will accept only `.gif` files with MIME type `image/gif` when the request `type` is `gif`.
- ZIP archives (Pixiv-style ugoira) are explicitly rejected. Uploaders must provide a GIF instead of a .zip archive.
- Still image posts (`illust`, `manga`, etc.) accept standard image formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`.

If you need to convert Ugoira archives to GIFs, perform conversion offline or in a preprocessing service before uploading.
