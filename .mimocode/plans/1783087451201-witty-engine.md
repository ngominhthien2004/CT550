# Remove chevron icon from sidebar collapse toggle

## Context
The sidebar collapse button (`AppSidebarMenu.vue:127`) shows a chevron-left/right icon. User considers it unnecessary — the button already has `aria-label` and `title` for clarity.

## File to modify
- `frontend/src/components/layout/AppSidebarMenu.vue` — line 127

## Change
Replace the conditional chevron icon with nothing (remove the `<i>` element entirely). The button's `aria-label` and `title` already communicate its purpose.

```vue
<!-- Before -->
<i :class="sidebarCompact ? 'fa-solid fa-chevron-right' : 'fa-solid fa-chevron-left'" aria-hidden="true"></i>

<!-- After -->
<!-- (remove the <i> element) -->
```

## Verification
- Navigate to any page → sidebar collapse button renders without icon
- Click toggle → sidebar still collapses/expands correctly
- `aria-label` still announces correctly for screen readers
